A minimal, high-performance universal acceleration proxy.

How to Use

This service can proxy and accelerate any URL. Usage is very simple:

{your_proxy_domain}/{target_url}

For example, to proxy GitHub resources, you can construct the URL like this:

https://proxy.sdjz.wiki/https://github.com/some/resource

This is very useful for accelerating access to slow foreign resources or solving network accessibility issues.

Available Nodes

Proxy currently has two nodes: JP (Tokyo, Japan) proxy.sdjz.wiki and HK (Hong Kong, China) hk.sdjz.wiki. You can choose the best node according to your network environment, data is now synchronized between nodes.
