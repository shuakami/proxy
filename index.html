Proxy V2 - 免费高性能代理加速服务

A minimal, high-performance universal acceleration proxy for GitHub, Google API, and global resources. 高性能免费代理服务，支持GitHub、Google API等全球资源加速访问。

代理使用教程 - How to Use Proxy Service

本代理服务可以加速任何URL访问，支持GitHub代理、Google API代理、NPM代理等。使用方法非常简单：This universal proxy service can accelerate any URL access, supporting GitHub proxy, Google API proxy, NPM proxy, etc. Usage is very simple:

{proxy_domain}/{target_url}

例如，要代理GitHub资源，可以这样构建URL。For example, to proxy GitHub resources, you can construct the URL like this:

https://proxy.sdjz.wiki/https://github.com/some/resource

这对于加速访问海外资源、解决网络访问问题、提高开发效率非常有用。支持CORS跨域、智能缓存、多节点负载均衡。This is very useful for accelerating access to overseas resources, solving network accessibility issues, and improving development efficiency. Supports CORS, intelligent caching, and multi-node load balancing.

Data is synchronized between nodes, providing 24/7 stable service.
